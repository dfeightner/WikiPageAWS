Resources to help study and pass the AWS Certified Solutions Architect Associate Exam


1. AWS Skill Builder

AWS provides valuable resources within AWS Skill Builder to help you study for their exams. They have a self-paced learning course that will help you learn about 90% of the exam. I highly recommend doing some more research into what will be on the last 10% of the exam. Here is the link to AWS Skill Builder. An acount is required to enroll in the courses. They also have labs and games, which help you prepare for the exam in a more hands-on approach.


2. LinkedIn Learning

LinkedIn Learing was a great resource for learning about AWS. I found LinkedIn learning videos to be quite helpful when studying for the exam. As a visual learner myself, I found the videos to be a nice visual and auditory learning experience. You can join LinkedIn learning through NextGen. You will be able to create a library of videos you've watched and are interested in.


3. Youtube

Another valuable resource, that sometimes gets overlooked, is Youtube. If I ever got confused on a concept, I would often resort to youtube because I would find easier to understand videos and intructions. I have attached a link to the video that helped me the most on youtube.


4. W3 Schools

This is a resource I used in college to find anything from coding help to practice exams. W3 schools is a helpful resource that I used to take practice exams. Here is the link to the W3 AWS homepage. I also found this link to be helpful as well.


5. Github

This is another resource I used while studying for the exam. While it's not as helpful as the other resources for learning, I used it as my own knowledge base for my notes.
